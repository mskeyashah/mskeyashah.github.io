At HackDavis 2020, my team and I built a mobile application called VeggieBuddie. This app won 2nd place in "Most Social Good" and "Best Use of Google Cloud" categories and was one of the top 20 teams in Google Solution Challenge 2020 where over 800 universities participated worldwide.

The app is published with enhancements on Google Play Store as Veg N' Snap.

Development

Process

  • Built the app on Android Studio using Flutter and Dart programming
  • Used Google user authentication to give user the advantage to save any dietary preferences, specifically allergens through their Google account. This data is stored in the Google cloud's realtime database, Firebase
  • The app implements Google's API for OCR to extract the text from the image of the ingredients
  • The text is then compared to the locally stored list of ingredients as well as the customized allergens list to determine a dietary restriction
  • The list of ingredients is updated automatically once there are changes to the online database
  • The user can also access the app offline by logging in through the guest account

Later on, another feature was added to the app which uploads any unknown ingredients to the Firebase database to improve the accuracy of the app. The app also displays the ingredients broken down into vegan, vegetarian, and non-vegetarian as well as allergens, if applicable.

I encourage you to use Veg N' Snap. I hope you find it useful if you have any dietary preferences. Kindly provide your feedback to help me improve it further. I appreciate it!

This hackathon was sponsored by HCL and Microsoft companies to encourage development of applications that could help in the fight against corona virus. My team and I created a mobile app called COVID Trace that allows the users to monitor their health, know their risk, and report symptoms. We were in the top 32 teams that qualified for the semifinals in more than 200 teams that participated in the hackathon.

Development

Process

  • Developed the app through Flutter packages and Dart programming language
  • Built the home screen with multiple resource links and an API that updates the global coronavirus count daily
  • Linked the self reporting page to the home screen where users can report their test results or contact and made sure to validate the answers
  • Integrated the calendar page that retrieved the corresponding data from the answers and displayed a note each day for the next 14 days reminding the user to self isolate and report any symptoms
  • The app includes prevention tips as well reliable sources such as CDC and WHO
  • All the data is anonymous and stored on the user's phone to avoid any breach in privacy

For UPenn Hacks 2020, my team and I built a web-based Machine Learning project called Accessible Smart Linguist that detects American Sign Language using the live video feed from the camera and implements features like autocorrect and autocomplete to reduce the communication time between two people. Furthermore, the Learn feature of our application enables users to explore and improve their sign language skills in a fun and engaging way.

The app won in "Best Use of Wolfram Technology" category at the hackathon.

Development

Process

  • Self-created an image dataset for the 26 letters of the alphabet using OpenCV and python scripts
  • Trained a Convolutional Neural Network using TensorFlow and Keras on the dataset for 50 epochs achieving about 98% accuracy
  • Integrated the model into our website through the use of Python and Flask
  • The website takes in a live video stream from the camera and the user can sign in the designated space
  • The last word is autocorrected and autocompleted every time the user enters a space. The user can also manually click the autocomplete button
  • The user can start or pause anywhere as well as start with a clean palette
  • The Autocorrect/Autocomplete features are implemented through python libraries using an NLP model that trains as the user types
  • The front-end of the application is developed through the use of HTML and Flask templating
